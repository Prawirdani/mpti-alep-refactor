### To Run:

#### Server:
- Buat DB Baru.
- Rename .env.example to `.env`, sesuaikan konfig DB local.
- Install Depedencies
```bash
npm install
```
- Run Migration
```bash
npm run migrate
```
- Run Seed 
```bash
npm run seed
```
- Run Server
```bash
npm run dev
```


#### Client:
- Install Depedencies
```bash
npm install
```
- Run Client
```bash
npm run dev
```
